# 星火篝火 · 新标签页番茄钟 — 产品需求文档

> 版本 1.0.0 · 与代码实现同步

## 产品定位

- **载体**：Chrome MV3 扩展，覆盖新标签页
- **体验**：Outer Wilds 风格 3D 篝火场景 + 经典番茄钟（25/5/15，4 轮一长休息）
- **仪式**：专注结束立即进入烤棉花糖近景（同双击篝火），同时开始休息倒计时

## 核心流程

1. **Idle**：显示当前时间，「开始专注」按钮
2. **Focus**：弱化时钟，显示 MM:SS 倒计时，可暂停/放弃
3. **专注完成**：立即进入烤火近景并开始短/长休息倒计时
4. **ShortBreak / LongBreak**：休息倒计时，可跳过；可手动退出烤火视角
5. 长休息结束轮次归零；可选自动开始下一轮

## 轮次规则

- `round` 始终为 **1 ~ cyclesBeforeLong** 的整数，表示当前周期内第几次专注
- 第 1 ~ (N−1) 次专注结束 → 短休息 → 轮次 +1
- 第 N 次专注结束 → 长休息（轮次保持 N）→ 长休息结束后轮次重置为 1
- 放弃专注、修改「几轮后长休息」、加载历史会话时自动校正非法轮次，避免出现「第 5/4 轮」

## 默认设置

| 参数 | 默认值 | 范围 |
|------|--------|------|
| 专注 | 25 min | 5–90 |
| 短休息 | 5 min | 1–30 |
| 长休息 | 15 min | 5–45 |
| 长休息轮数 | 4 | 2–8 |

## UI 状态

| 状态 | 时钟 | 主显示 | 按钮 |
|------|------|--------|------|
| Idle | 100% | 当前时间 | 开始专注 |
| Focus | 35% | 倒计时 | 暂停、放弃 |
| Paused | 35% | 冻结时间 | 继续、放弃 |
| Short/Long Break | 35% | 倒计时（可叠加烤火近景） | 跳过休息 |

## 技术架构

```
newtab.html → newtab-main.js
  ├── pomodoro.js（状态机 + storage）
  ├── pomodoro-ui.js（HUD / 设置 / 引导）
  ├── confirm-dialog.js（统一确认弹窗）
  ├── audio.js（MP3 提示音 + 休息背景音乐）
  ├── app.js（Three.js 场景）
  └── background.js（alarms + notifications）
```
